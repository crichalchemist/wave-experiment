# Detective LLM — Knowledge Vault

This vault serves as the project's persistent memory. It is both a decision log (consulted during development) and a runtime knowledge base (queried by the system during gap analysis).

## Structure

| Directory | Contents |
|---|---|
| `decisions/` | Architecture Decision Records (ADRs) — why each significant choice was made |
| `hypothesis-traces/` | Records of hypothesis evolution during analysis runs |
| `gap-findings/` | Substantive gap findings from each analysis run |
